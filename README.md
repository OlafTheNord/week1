# week1
Первая неделя обучения (базовый GO)

### Задача:
Напиши программу, которая:

 - Читает JSON-файл с данными о 5 сотрудниках (name, department, salary, hire_date)
 - Вычисляет среднюю зарплату по отделам
 - Выводит результат в виде map[string]float64
 - Обрабатывает ошибки чтения файла и невалидных данных (не паникуй!)
 - Использует только стандартную библиотеку (encoding/json, os, fmt, time)
